# 人脸识别与健康关联分析系统使用说明

## 运行方式

1. 打开命令行，进入项目目录：

```powershell
cd "C:\Users\zhangkai\Desktop\A-人脸识别与健康关联分析设计与实现"
```

2. 启动后端服务：

```powershell
npm start
```

也可以直接运行：

```powershell
node --no-warnings server.js
```

3. 浏览器访问：

```text
http://127.0.0.1:8000/
```

## 用户端流程

用户端从首页开始：

首页管理.html -> 用户登录 login.html -> 首页管理.html -> 用户中心界面.html

## 已实现功能

- 页面之间可跳转：首页、用户管理、人脸识别记录、健康数据、健康报告、公告、轮播图、用户中心、登录、注册。
- 已新增管理员登录界面和管理端口页面。
- 使用 Node.js 后端提供接口服务。
- 使用 SQLite 数据库文件 `app.db` 保存数据。
- 支持用户注册、登录、用户新增、编辑、删除。
- 用户中心可读取并修改当前用户的个人信息。
- 后端已建立用户、健康报告、人脸识别记录、健康数据、公告、轮播图等数据表。

## 主要接口

```text
GET    /api/health          后端连通性检查
GET    /api/users           查询用户列表
GET    /api/users/:id       查询单个用户
POST   /api/register        用户注册
POST   /api/login           用户登录
POST   /api/admin-login     管理员登录
POST   /api/users           新增用户
PUT    /api/users/:id       修改用户
DELETE /api/users/:id       删除用户
GET    /api/stats           查询统计数据
```

## 数据库说明

数据库文件位于：

```text
C:\Users\zhangkai\Desktop\A-人脸识别与健康关联分析设计与实现\app.db
```

第一次启动 `server.js` 时会自动创建数据库和基础测试数据，不需要手动建库。

主要数据表：

```text
users              用户信息表
health_reports     健康报告表
face_records       人脸识别记录表
health_data        健康数据表
announcements      公告表
carousel_items     轮播图表
```

## 注意事项

- 不要直接双击 HTML 文件使用数据库功能，必须先启动 `node server.js`。
- 如果 8000 端口被占用，可以换端口启动：

```powershell
$env:PORT=8010; npm start
```

然后访问：

```text
http://127.0.0.1:8010/
```
