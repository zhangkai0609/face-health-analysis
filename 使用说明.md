## 公网访问（在别的电脑上访问本系统）

本项目可以**复制整个文件夹到任何一台有 Node.js 的电脑**上运行，并生成公网链接让其他人访问。

### 在别的电脑上部署步骤

1. **安装 Node.js**（如果还没有）
   - 下载地址：https://nodejs.org/ （下载 LTS 版本）
   - 安装完成后打开命令行验证：`node -v`

2. **复制项目到该电脑**
   - 把整个 `A-人脸识别与健康关联分析设计与实现` 文件夹复制过去即可
   - **不需要** `npm install`（项目零依赖，纯 Node.js 内置模块）

3. **启动服务并生成公网链接**
   - 进入项目文件夹，**双击** `公网启动.bat`
   - 也可以双击 `启动公网访问.bat`，这两个文件现在都会调用同一个稳定启动脚本
   - 窗口会显示类似 `https://xxxx.trycloudflare.com` 的公网地址
   - 同时会生成 `公网访问地址.txt`，里面也保存了公网地址、管理员入口和账号密码
   - 对方在任何地方打开这个 HTTPS 地址就能访问系统

4. **停止服务**
   - 双击 `公网停止.bat`
   - 关闭后，公网地址就不能继续访问

### 跨电脑访问说明

| 场景 | 方法 | 要求 |
|------|------|------|
| 同一 WiFi | 直接用 `http://电脑IP:8000` | 在同一局域网 |
| 不同网络 | 运行 `公网启动.bat` 生成公网链接 | 电脑需能访问外网 |
| 任何地方 | 使用生成的 `https://xxx.trycloudflare.com` 链接 | 无需同一网络 |

### 数据文件

- 数据库文件 `app.db` 会保存在项目目录下
- 复制整个项目文件夹到新电脑后，之前的数据也会一起带过去
- 如果不需要旧数据，删除 `app.db` 文件，重启服务会自动创建新的

## 注意事项

- 不要直接双击 HTML 文件使用数据库功能，必须先启动 `node server.js`。
- 如果 8000 端口被占用，可以换端口启动：

```powershell
$env:PORT=8010; npm start
```

然后访问：

```text
http://127.0.0.1:8010/
```
