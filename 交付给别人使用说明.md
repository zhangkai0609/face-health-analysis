# 交付给别人使用说明

## 一、你需要发给对方什么

把整个文件夹打包发给对方：

```text
A-人脸识别与健康关联分析设计与实现
```

不要只发某一个 HTML 文件。必须整个文件夹一起发，因为里面包含：

```text
server.js              后端服务
app.js                 前端交互脚本
app.db                 数据库
tools/cloudflared.exe  公网访问工具
公网启动.bat            一键启动
公网停止.bat            一键停止
所有 HTML 页面
```

## 二、对方电脑需要准备什么

对方电脑需要安装：

```text
Node.js 24 或更新版本
```

下载地址：

```text
https://nodejs.org/
```

安装完成后，重新打开项目文件夹即可。

## 三、对方怎么启动

1. 解压你发给他的整个项目文件夹。
2. 双击：

```text
公网启动.bat
```

3. 等待窗口出现“公网访问地址”。
4. 复制这个地址发给别人，例如：

```text
https://xxxx.trycloudflare.com
```

同时脚本会自动生成：

```text
公网访问地址.txt
```

打开这个 txt 也能看到公网地址、管理员入口、账号密码。

## 四、别人怎么访问

普通访问：

```text
https://xxxx.trycloudflare.com
```

管理员入口：

```text
https://xxxx.trycloudflare.com/管理员登录界面.html
```

管理员账号：

```text
账号：admin
密码：admin123456
```

用户测试账号：

```text
用户名：zhangsan
密码：123456
```

## 五、怎么关闭

双击：

```text
公网停止.bat
```

关闭后，公网地址就不能访问了。

## 六、重要说明

- 公网地址是临时地址，每次重新双击 `公网启动.bat` 后可能会变化。
- 对方电脑必须保持开机、联网，别人才能访问。
- 数据保存在项目里的 `app.db`，换电脑后使用的是那台电脑文件夹里的数据库。
- 不要直接双击 HTML 文件运行，必须使用 `公网启动.bat` 或 `node server.js` 启动服务。
